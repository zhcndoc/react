2020 年是漫长的一年。随着这一年接近尾声，我们想分享一篇关于我们对零包体积 **React Server Components** 研究的特别节日更新。

</Intro>

---

为了介绍 React Server Components，我们准备了一场演讲和一个演示。如果你愿意，可以在假期期间看看，或者等到新的一年工作重新开始后再看。

<YouTubeIframe src="https://www.youtube.com/embed/TQQPAU21ZUw" />

**React Server Components 仍处于研究和开发阶段。** 我们分享这项工作是出于透明的精神，也希望从 React 社区获得初步反馈。以后会有充足的时间去做，所以**不要觉得你现在必须立刻跟上！**

如果你想了解它们，我们建议按以下顺序进行：

1. **观看演讲**，了解 React Server Components 并观看演示。

2. **[克隆演示](http://github.com/reactjs/server-components-demo)**，在你的电脑上体验 React Server Components。

3. **[阅读 RFC（末尾有 FAQ）](https://github.com/reactjs/rfcs/pull/188)**，以获得更深入的技术解析并提供反馈。

我们很期待在 RFC 中或通过回复 [@reactjs](https://twitter.com/reactjs) 的 Twitter 账号听到你的声音。节日快乐，注意安全，我们明年见！
