[在 10 月](/blog/2025/10/07/introducing-the-react-foundation)，我们宣布了组建 React 基金会的计划。今天，我们很高兴地宣布 React 基金会已正式启动。

React、React Native 以及像 JSX 这样的支持项目不再归 Meta 所有——它们现在归 React 基金会所有，这是一个由 Linux 基金会托管的独立基金会。你可以在 [Linux 基金会的新闻稿](https://www.linuxfoundation.org/press/linux-foundation-announces-the-formation-of-the-react-foundation) 中了解更多信息。

### 创始成员 {/*founding-members*/}

React 基金会有八家白金级创始成员：**Amazon**、**Callstack**、**Expo**、**Huawei**、**Meta**、**Microsoft**、**Software Mansion** 和 **Vercel**。自 [我们在 10 月的公告](/blog/2025/10/07/introducing-the-react-foundation) 发布以来，**Huawei** 已加入。React 基金会将由董事会治理，董事会由各成员代表组成，[Seth Webster](https://sethwebster.com/) 将担任执行主任。

### 新的临时领导委员会 {/*new-provisional-leadership-council*/}

React 的技术治理将始终独立于 React 基金会董事会——React 的技术方向将继续由为 React 做出贡献并维护它的人来决定。我们已成立一个临时领导委员会来确定这一结构。我们将在未来几个月内分享更新。

### 下一步 {/*next-steps*/}

要完成这次过渡，还有工作要做。在未来几个月里，我们将：

* 最终确定 React 的技术治理结构
* 将仓库、网站和其他基础设施转移到 React 基金会
* 探索支持 React 生态系统的项目
* 启动下一届 React Conf 的筹备工作

随着这些工作的推进，我们会分享最新进展。

### 感谢 {/*thank-you*/}

如果没有过去十年中塑造 React 的成千上万名贡献者，这一切都不可能实现。感谢我们的创始成员，感谢每一位提交过 pull request、报告过 issue 或帮助他人学习 React 的贡献者，也感谢每天使用 React 构建应用的数百万开发者。React 基金会之所以存在，正是因为这个社区，我们期待与大家一起构建它的未来。
