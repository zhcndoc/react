## 如何贡献 {/*how-to-contribute*/}

你可以为翻译工作做出贡献！

社区在 react.dev 的各个语言分支上为 React 文档开展翻译工作。典型的翻译工作包括直接翻译 Markdown 文件并创建拉取请求。点击上方指向你所用语言 GitHub 仓库的“贡献”链接，并按照其中的说明帮助完成翻译工作。

如果你想为你的语言开始一个新的翻译，请访问：[translations.react.dev](https://github.com/reactjs/translations.react.dev)
